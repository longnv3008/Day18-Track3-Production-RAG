# Judge Bias Report

| Bias | Measurement | Observation | Mitigation |
|---|---:|---|---|
| Position bias | A wins first position 10/30 (33.3%) | Swap-and-average reduces order preference. | Always run swapped pairwise judging. |
| Length bias | Longer B wins 3/20 (15.0%) | Longer answers can look more helpful. | Penalize verbosity through conciseness rubric. |

Conclusion: keep swap-and-average for pairwise judging and use absolute scoring dimensions to separate accuracy from style.
